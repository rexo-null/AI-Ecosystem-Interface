# AI Ecosystem Interface

Собственная IDE с интеграцией локальной нейросети Qwen 2.5 14b Coder через llama.cpp.

## Архитектура

- **Client (Frontend)**: Monaco Editor + чат-интерфейс
  - Древовидная структура проекта (слева)
  - Редактор кода с вкладками (центр)
  - Чат для взаимодействия с агентом (справа)

- **Server (Backend)**: 
  - API для работы с файловой системой
  - Интеграция с llama.cpp для запуска Qwen 2.5 14b Coder
  - Управление системной консолью

## Функционал

- Чтение, редактирование, удаление файлов
- Навигация по файловой системе
- Интеграция с локальной нейросетью
- Системная консоль
- Агентное взаимодействие

## Технологии

- Frontend: Monaco Editor, React/Vue (на выбор)
- Backend: Node.js/Python
- AI: llama.cpp + Qwen 2.5 14b Coder

## Установка

```bash
# Установка зависимостей
npm install

# Запуск разработки
npm run dev
```
